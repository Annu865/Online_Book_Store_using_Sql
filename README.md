# ЁЯУЪ Online Bookstore Sales Analysis Using SQL

## About The Project

Data is useful only when it helps answer real business questions.

In this project, I analyzed an online bookstore database using SQL to understand customer purchasing behavior, sales performance, inventory status, and revenue trends.

The project simulates real-world business scenarios where a bookstore owner wants to identify top customers, best-selling books, high-performing genres, and stock availability to support better business decisions.

---

## Project Objectives

The main goals of this project were:

- Analyze book sales performance
- Identify top customers based on spending
- Discover best-selling books and genres
- Monitor inventory and remaining stock
- Generate revenue insights
- Practice real-world SQL analysis techniques

---

## Database Structure

The project uses three related tables:

### Books
Stores information about books available in the bookstore.

### Customers
Contains customer information including location details.

### Orders
Stores transaction records connecting customers and books.

## Business Questions Answered

Some of the questions explored in this analysis include:

- Which books belong to a specific genre?
- Which books were published after a certain year?
- Which customers generated the highest revenue?
- What is the total revenue generated by the bookstore?
- Which books are ordered most frequently?
- Which genres contribute the most to sales?
- Which customers place repeat orders?
- Which books require restocking?
- How much revenue does each author generate?
- Which customers spend above average?

---

## Key Findings

### Revenue Analysis
Calculated overall bookstore revenue and identified major revenue contributors.

### Customer Analysis
Identified repeat customers and top spenders.

### Product Analysis
Discovered the most frequently ordered books and best-performing genres.

### Inventory Analysis
Tracked remaining stock after sales to highlight books that may require replenishment.
